# BadWordFilter

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bad_word_filter'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install bad_word_filter

## Usage
content = "This is great text nothing wrong here"
```BadWordFilter.filter(content)```
response will be in json
